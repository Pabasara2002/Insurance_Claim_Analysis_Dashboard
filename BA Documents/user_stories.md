## User Stories

**1. Analyze Claim Distribution**  
As an Insurance Manager, I want to analyze total claims by category so that I can identify high-risk areas.

**2. Identify High-Risk Customers**  
As a Risk Analyst, I want to analyze customer demographics so that I can predict high-risk profiles.

**3. Monitor Claim Trends**  
As a Business Analyst, I want to track claim trends over time so that I can identify patterns.

**4. Evaluate Coverage Zones**  
As an Insurance Manager, I want to compare claims by location so that I can adjust policies accordingly.
